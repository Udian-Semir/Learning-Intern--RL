## 组件详细说明

### 1. Pons Adapter (特征压缩器)

**功能**: 将来自 VLM 的长序列多层隐藏状态压缩为固定长度的特征向量。

**架构**:
- **输入**: 多层 VLM 隐藏状态 `List[(B, seq_len, D)]`
- **合并**: 沿 seq_len 维度拼接 → `(B, seq_len × num_layers, D)`
- **Query Tokens**: 可学习的查询向量 `(1, Q_len, D)` + 位置编码
- **Cross-Attention Blocks**: N 个交叉注意力块
  - Query 查询合并后的 VLM 特征
  - 每个块包含: Cross-Attention → LayerNorm + Residual → FFN → LayerNorm + Residual
- **输出**: `(B, Q_len, D)` - 压缩后的特征

**默认配置**:
```python
q_seq_len = 64          # Query 序列长度
num_blocks = 2          # Cross-Attention 块数
num_heads = 8           # 注意力头数
dropout = 0.1           # Dropout 概率
ffn_expansion = 4       # FFN 扩展倍数
```

### 2. ParaCAT Action Head (动作预测头)

**功能**: 从 Pons 输出预测离散化的机器人动作。

**架构**:
- **输入**: Pons 输出 `(B, Q_len, D)`
- **Action Query Tokens**: 可学习的动作查询 `(1, chunk_size × action_dim, D)` + 位置编码
- **Cross-Attention Block**: 查询 Pons 输出
- **Self-Attention Blocks**: M 个自注意力块 (无 mask，全局注意力)
- **MLP Layers**: 多层感知机
  - 中间层: GELU 激活
  - 最后一层: 无激活函数，输出 3 类 logits
- **Reshape**: `(B, C×A, 3)` → `(B, C, A, 3)`
- **输出**: `(B, chunk_size, action_dim, 3)` - 3 分类 logits

**默认配置**:
```python
chunk_size = 16                # 动作序列长度
action_dim = 7                 # 动作维度 (如 6DOF + gripper)
num_transformer_blocks = 2     # Self-Attention 块数
num_mlp_layers = 2             # MLP 层数
mlp_expand_dim = 1024          # MLP 隐藏层维度
num_heads = 8                  # 注意力头数
```

### 3. 离散化输出映射

**输出解释**:
```
Output Shape: (Batch, ChunkSize, ActionDim, 3)

最后一维 (3 classes):
  - logits[..., 0] → Class 0 → -1 (后退)
  - logits[..., 1] → Class 1 →  0 (不动)
  - logits[..., 2] → Class 2 → +1 (前进)

推理流程:
  1. class_idx = argmax(logits, dim=-1)    # {0, 1, 2}
  2. discrete_val = class_idx - 1          # {-1, 0, +1}
  3. continuous_action = discrete_val × delta
```

## 训练配置示例

```python
# 模型配置
config = {
    # Pons Adapter
    'pons_q_seq_len': 64,
    'pons_num_blocks': 2,
    'pons_num_heads': 8,
    'pons_dropout': 0.1,
    
    # ParaCAT Action Head
    'chunk_size': 16,
    'action_dim': 7,
    'num_transformer_blocks': 2,
    'num_mlp_layers': 2,
    'mlp_expand_dim': 1024,
    
    # VLM
    'vlm_output_dim': 2560,     # Qwen2.5-3B
    'num_vlm_layers': 3,        # 使用 3 层 hidden states
    
    # 训练
    'batch_size': 32,
    'lr': 1e-4,
    'steps': 10000,
    'discrete_actions': True,
}
```

## 参数量估算

|------|--------|

| Pons Position Encoding | ~164K |
| Pons Cross-Attention Blocks (×2) | ~53M |
| ParaCAT Action Queries | ~287K |
| ParaCAT Position Encoding | ~287K |
| ParaCAT Cross-Attention | ~26M |
| ParaCAT Self-Attention (×2) | ~53M |
| ParaCAT MLP Layers | ~5M |
| **总计** | **~140M** |

## 优势

1. **高效特征压缩**: Pons 将长序列 VLM 特征压缩为固定长度
2. **离散化动作**: 3 分类输出简化学习任务，减少预测误差累积
3. **并行预测**: 一次性预测整个 chunk 的动作序列
4. **灵活配置**: 支持不同的 VLM 隐藏层组合
5. **端到端训练**: Pons 和 ParaCAT 可联合优化
